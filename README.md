# Analyse des municipales 2026  
**Corrélation entre niveau de vie communal et participation électorale**  
Projet réalisé dans le cadre du *Challenge Open Data*.

---

## Contenu du projet
Ce dépôt contient :

- Un **notebook** présentant l’ensemble du code utilisé pour l’analyse.
- Un **rapport HTML** offrant une lecture plus fluide et illustrée des résultats.

---

## Consulter le rapport
Pour une lecture optimale, il est recommandé d’ouvrir la **version HTML** du rapport :

1. Télécharger le fichier `.html` sur votre machine  
2. Double‑cliquer pour l’ouvrir dans votre navigateur

---

## Données utilisées
Les jeux de données proviennent de **data.gouv.fr** :

- `BASE_TD_FILO_DISP_IRIS_2018`
- `general_results`

Pour exécuter le notebook, placez ces fichiers dans un dossier `sample_data/`.

---

## Exécuter le notebook
Vous aurez besoin :

- d’un environnement Python (Jupyter, VS Code, google collab, etc.)
- des bibliothèques usuelles pour l’analyse de données (pandas, numpy, matplotlib ou plotly selon vos besoins)
